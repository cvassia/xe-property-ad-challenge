# XE Property Ad Challenge

A full-stack implementation of the XE Web Developer Challenge for creating real estate ads, with a strong focus on area autocomplete, validation, persistence, responsive UI, and tests.


- React + TypeScript + Vite for the frontend
- Node.js + Express + TypeScript for the backend
- SQLite 
- Client-side and server-side validation with Yup
- Autocomplete backend proxy with caching
- Vitest, React Testing Library, Supertest, and Playwright-ready structure

## Getting started

```bash
npm install
npm run dev
```
